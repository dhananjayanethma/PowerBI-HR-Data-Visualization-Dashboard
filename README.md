## Project Overview

The dashboard is designed to answer key HR questions such as:

-  Employee attendance and working presence
-  Weekly and monthly attendance trends
-  Work From Home (WFH) vs Office work patterns
-  Identification of seasons or weeks with high WFH activity

The insights generated can help HR departments improve **workforce planning, productivity analysis, and policy decisions**.
